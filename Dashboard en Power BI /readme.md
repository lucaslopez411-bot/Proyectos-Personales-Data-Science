# Dashboard Financiero y Comercial

## Objetivo

Desarrollar un dashboard interactivo para monitorear ventas,
rentabilidad, márgenes y KPIs operativos.

## Herramientas

- Power BI
- SQL
- Excel
- DAX

## KPIs Analizados

- Ventas
- Margen Bruto
- Rentabilidad
- Ticket Promedio
- Inventario

## Principales Insights

- Identificación de productos con menor margen.
- Análisis de estacionalidad.
- Seguimiento de rentabilidad por categoría.
- Detección de oportunidades de optimización.
